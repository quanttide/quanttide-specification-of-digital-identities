# 权限

## 常见权限设计模型

- 自主访问控制（DAC：Discretionary Access Control）
- 强制访问控制（MAC：Mandatory Access Control）
- 访问控制列表（ACL：Access Control List）
- 基于角色的访问控制（RBAC：Role-Based Access Control）
  - RBAC0
  - RBAC1
  - RBAC2
  - RBAC3
- 基于属性的权限验证（ABAC：Attribute-Based Access Control）
